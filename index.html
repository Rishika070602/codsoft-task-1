Line following and obstacle avoiding robot

The Line Following and Obstacle Avoiding Robot is an autonomous vehicle designed to follow a black line on a white surface using infrared sensors while intelligently detecting and avoiding obstacles with an ultrasonic sensor. Controlled by an Arduino microcontroller, the robot processes real-time sensor data to adjust its movement, ensuring it stays on track and navigates around obstacles without human intervention. This project demonstrates the integration of sensor-based navigation, basic decision-making algorithms, and embedded system programming, making it a great introduction to robotics and automation.
